# Virtual Startup Team

## How It Works

You talk to me (r1, the orchestrator). I spawn specialized subagents for tasks.

## The Team

| Role | Focus | Spawn Keyword |
|------|-------|---------------|
| CEO | Strategy, decisions | `sessions_spawn` with task to "CEO agent" |
| PM | Product, requirements | `sessions_spawn` with task to "PM agent" |
| Engineer | Code, architecture | `sessions_spawn` with task to "Engineer agent" |
| Designer | UI/UX, brand | `sessions_spawn` with task to "Designer agent" |
| Marketer | Growth, content | `sessions_spawn` with task to "Marketer agent" |
| CSM | Customers, feedback | `sessions_spawn` with task to "CSM agent" |

## Example Commands

- "PM, research the market for AI tools for developers"
- "Engineer, outline a technical architecture for X"
- "Marketer, create a launch strategy"
- "CEO, prioritize these 5 ideas"
