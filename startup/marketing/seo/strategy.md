## On-Page Optimization Checklist

### Page-Level SEO

#### Title Tag
- [ ] Include primary keyword
- [ ] Keep under 60 characters
- [ ] Make it compelling (click-worthy)
- [ ] Include brand name for homepage

**Examples:**
- `Transaction Coordinator Software | [Brand]`
- `Best AI Transaction Coordinator for Real Estate | [Brand]`
- `How to Never Miss a Deadline in Real Estate | [Brand]`

#### Meta Description
- [ ] Include primary keyword naturally
- [ ] Keep under 160 characters
- [ ] Include a call to action
- [ ] Make it compelling and unique

**Examples:**
- `Discover the best transaction coordinator software for real estate agents. Automate deadlines, client updates, and document management. Start free trial.`
- `Learn how to never miss a real estate deadline again. Our complete guide covers tools, tips, and automation strategies for agents.`

#### URL Structure
- [ ] Short and descriptive
- [ ] Include primary keyword
- [ ] Use hyphens, not underscores
- [ ] No special characters or parameters

**Examples:**
- `/transaction-coordinator-software`
- `/ai-transaction-coordinator`
- `/never-miss-deadline-real-estate`

#### Header Structure
- [ ] Single H1 with primary keyword
- [ ] H2s for major sections (include keywords)
- [ ] H3s for subsections
- [ ] Logical hierarchy

**Example:**
```
H1: Transaction Coordinator Software: Complete Guide
H2: What is Transaction Coordinator Software?
H2: Best Transaction Coordinator Tools
H3: [Tool 1]
H3: [Tool 2]
H2: How to Choose the Right Software
H3: Features to Look For
H3: Pricing Comparison
H2: Benefits of Transaction Coordinator Software
```

#### Content
- [ ] Primary keyword in first 100 words
- [ ] Natural keyword density (1-2%)
- [ ] Related keywords and synonyms
- [ ] Minimum 1,500 words for pillar pages
- [ ] Minimum 1,000 words for blog posts
- [ ] Short paragraphs (3-4 sentences)
- [ ] Bullet points and numbered lists
- [ ] Internal links to related content
- [ ] External links to authoritative sources
- [ ] Images with alt text

#### Images
- [ ] Descriptive file names
- [ ] Alt text with keywords (where natural)
- [ ] Compressed file sizes (< 100KB)
- [ ] Responsive images
- [ ] Image schema if applicable

#### Internal Linking
- [ ] Link to 3-5 related internal pages
- [ ] Use descriptive anchor text
- [ ] Link from high-authority pages to new content
- [ ] Create content clusters around topics

#### Schema Markup
- [ ] Organization schema on homepage
- [ ] SoftwareApplication schema on product pages
- [ ] Article schema on blog posts
- [ ] FAQ schema on FAQ pages
- [ ] Breadcrumb schema

### Technical SEO

#### Site Speed
- [ ] Core Web Vitals pass
- [ ] LCP < 2.5s
- [ ] FID < 100ms
- [ ] CLS < 0.1
- [ ] Images optimized
- [ ] CSS/JS minified
- [ ] Gzip compression enabled

#### Mobile Optimization
- [ ] Mobile-first design
- [ ] Responsive layout
- [ ] Touch-friendly navigation
- [ ] Fast mobile load times
- [ ] No intrusive interstitials

#### Crawlability
- [ ] XML sitemap submitted
- [ ] Robots.txt configured
- [ ] No orphan pages
- [ ] Clean URL structure
- [ ] Canonical tags set

#### Indexation
- [ ] No duplicate content
- [ ] Meta robots set correctly
- [ ] Pagination handled
- [ ] Parameter URLs excluded
